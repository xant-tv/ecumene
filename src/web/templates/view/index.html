Ecumene

Discord Bot - Destiny Integration - Clan Management

About

Ecumene is a Discord bot that provides clan management services for Destiny 2 through a series of simple commands. Unlike other popular tools, this bot empowers clan leaders with the ability to monitor member activity and automate approvals or removals.

Supported functionality:

  • Slash commands with role-based restrictions.
  • Clan activity monitoring and querying.
  • Kick from clan directly through Discord.
  • Automatic notification on pending member requests.

With a bit of savvy, you can host your own instance of this website and bot.
Please download the source code or contact the developers for help.
